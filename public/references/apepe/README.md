# APEPE Reference Images

Drop your official APEPE character images here. These are passed to the AI as reference for character consistency.

## Requirements

- **Format**: PNG or JPG
- **Resolution**: 1024x1024 or higher
- **Count**: 3 to 5 images recommended
- **Naming**: `apepe-01.png`, `apepe-02.png`, `apepe-03.png`, ...

## Best Practices

- Front-facing or 3/4 view shots
- Clean / minimal background
- Clear character visibility
- Different poses if possible (front, side, action)
